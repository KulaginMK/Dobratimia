# Звуки

| Папка | Назначение | Файлы |
|-------|------------|--------|
| `scream/` | «Отпустить» | `release.mp3`, `whoosh.mp3` |
| `ambient/` | Подкасты природы | `rain.mp3`, `wind.mp3`, `forest.mp3`, `waves.mp3`, `fire.mp3` |
| `ui/` | Клики интерфейса | `click.mp3` |

Без MP3 приложение использует синтезированный звук (Web Audio API) для «Отпустить».
