# Добратимия (React)

Vite + React + TypeScript + Tailwind CSS v4 + React Router.

## Команды

```bash
npm install
npm.cmd run dev      # PowerShell: npm.cmd если блокируется npm.ps1
npm.cmd run build
npm.cmd run preview
```

## Структура

- `src/features/` — страницы
- `public/data/` — JSON (анатомия стресса)
- `public/assets/` — звуки, изображения, будущие 3D-модели
